# Curve 2d
simple 2d curve library supporting beziers, catmulls, arcs,
lines and any combinations thereof. Also contains some vec2d utils.


## Installing
```
npm i @rdme/curve-2d
```
```
yard add @rdme/curve-2d
```
```
pnpm i @rdme/curve-2d
```
